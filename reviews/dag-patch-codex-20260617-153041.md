Short verdict: the patch is sufficient in spirit and does not overshoot. I would make one material tweak: add `Ideology -> M`, and relabel `Sn` away from “exogenous” toward “non-policy composition sources.”

1. **Does it resolve the two critiques?**  
Yes, mostly. `T -> C` plus `Sn -> C` fixes the old mistake of treating all composition as a mediator. `pi -> f`, `C -> f`, and `M -> f` fixes the measurement collapse and gives the rung-3 transcription-convention shock a home: `M` can move `f` while `pi` stays fixed.

What remains slightly wrong: calling `Sn` “exogenous” and then calling the `Sn -> C -> f` path “confounding” is too strong in a formal DAG. It is a bias source unless `Sn` is aligned with treatment timing, group, or ideology. Better label it `S_C`: “non-policy composition sources,” not necessarily independent.

2. **8 nodes / 10 edges?**  
This is the right scale. Do not fold `Sn` into `C`; that would undo the main repair. The extra node is exactly what lets the caption say: some composition is policy-moved, some is not.

I would tolerate 11 edges with `Ideology -> M`. I would not add the full time-expanded graph or split every spillover channel in the figure.

3. **Should `M` be root?**  
I would include `Ideology -> M`.

Recommended main edge list:

```text
Ideology -> T
Ideology -> Tn
Ideology -> pi
Ideology -> M        [add]
T -> pi              [target effect]
T -> C               [policy-moved composition]
S_C -> C             [non-policy composition source]
pi -> f
C -> f
M -> f
Tn -> f              [dashed shorthand]
```

Trade-off: adding `Ideology -> M` makes the graph slightly busier, but it captures the sharpest rung-3 danger: the same gender-ideology wave can change transcription/coding norms and create an apparent effect in `f` with no change in usage. Root `M` is fine for random archive or coding accidents; it is too weak for this paper’s own example.

If the studied policy directly changes official recording practice, the strict graph would also allow `T -> M -> f`. I would probably keep that in prose unless the caption’s measurement warning needs to be maximally explicit.

4. **Does `Sn -> C` + `T -> C` work?**  
Yes, if `Sn` is understood as “non-policy composition sources,” not as an independent random root. It covers archive coverage, outlet entry, author turnover, register mix, topic/referent mix, and speaker-pool drift.

I would not add `Sn -> f` unless `Sn` includes something that affects measurement directly. If archive/OCR/transcription quality changes the observed count without changing composition, that is `M -> f`, not `Sn -> f`.

`Ideology -> C` is substantively plausible: ideology can change who gets published, who speaks, what offices women hold, and what topics appear. But I would not draw it if the caption says `S_C` includes non-policy, possibly ideology-correlated composition sources. If the node remains labelled “exogenous,” then yes, a referee can fairly ask for `Ideology -> C` or `Ideology -> S_C`.

`Sn/S_C` should be treated as partly observed. Some margins are measurable and reweightable; the residual is unobserved and belongs in sensitivity/failure-map language.

5. **Keep `Tn -> f` shorthand?**  
Yes. With caption support, the dashed `Tn -> f` edge is defensible as compression.

The expanded version would be:

```text
Tn -> pi
Tn -> C
Tn -> M
```

But I would not use that in the main figure. It risks making the figure about spillover mechanics rather than the core lesson: neighbour policy contaminates the observed series through several routes.

6. **What would a sharp referee still catch?**  
Three things.

First, “confounder” language around `S_C` needs care. In this compact static graph, `S_C -> C -> f` is not formally a back-door path from `T` unless its timing/group pattern is aligned with treatment. Say “non-policy composition drift/bias source” unless the alignment is explicit.

Second, blanket “adjust for C” language is still dangerous. Since `T -> C` exists, reweighting on treatment-moved margins subtracts part of the effect. The caption should say: adjust/reweight only on pre-specified non-policy margins; otherwise diagnose and report.

Third, if `M` has no parent, the graph understates the rung-3 threat. Add `Ideology -> M`; that is the one change I would make.
